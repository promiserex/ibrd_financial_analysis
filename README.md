## Key Financial Note

The **disbursed amount** represents the total money the IBRD has ever released to borrowers. Over time, many countries have repaid part of what they borrowed. These repayments reduce the **Loans Held (outstanding balance)**, but the original disbursed total remains unchanged. This is why **disbursed values are higher than current outstanding loan balances**.
